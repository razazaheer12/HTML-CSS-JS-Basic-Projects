# Life Clock Meter

A simple and elegant web application that calculates your remaining life in days, weeks, and months, assuming a lifespan of 90 years. Built with vanilla HTML, CSS, and JavaScript for a clean, responsive experience.

## 🌟 Features

- **Life Calculation**: Estimates remaining days, weeks, and months based on your current age.
- **Input Validation**: Ensures age input is between 0 and 90 years.
- **Responsive Design**: Optimized for desktop and mobile devices.
- **Clean UI**: Card-based layout with smooth transitions and a modern look.
- **Google Fonts Integration**: Uses 'Fredoka' font for a friendly, rounded appearance.
- **Instant Results**: Real-time calculation on button click.

## 🚀 Demo

Check out the live demo: [Life Clock Meter](https://life-clock-meter-12.netlify.app/) 

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/LifeClock.git
   ```

2. Navigate to the project directory:
   ```bash
   cd LifeClock
   ```

3. Open `index.html` in your preferred web browser.

No additional dependencies or build steps required!

## 💡 Usage

1. Open the application in your browser.
2. Enter your current age in the input field (must be between 0 and 90).
3. Click the "Calculate" button.
4. View your remaining life in days, weeks, and months.

## 🛠️ Technologies Used

- **HTML5**: Structure and semantic markup.
- **CSS3**: Styling with responsive design and transitions.
- **JavaScript (ES6+)**: Logic for calculations and input validation.
- **Google Fonts**: 'Fredoka' for a cute, rounded typography.
