## 🎨 Project Overview

This project is part of the **Mini Web Projects** series — a collection of modern, lightweight UI components built to showcase clean code and beautiful design.

The pricing table features:
- A sophisticated **dark aesthetic** with deep background (#0d0d14)
- **Glassmorphism-inspired** shadows and glowing hover effects
- **Smooth animations** on hover (lift + glow)
- **Fully responsive** layout that adapts to all screen sizes

## 🚀 How to Run

### 1. Clone the Repository

```
bash
git clone https://github.com/razazaheer12/pricing-table.git
cd pricing-table
```

### 2. Open in Browser

Simply open the `index.html` file in any modern web browser:

```
bash
# For Windows
start index.html

# For macOS
open index.html

# For Linux
xdg-open index.html
```

Or drag and drop `index.html` into your browser.

## 💡 Customization

Want to make it your own? Here are some quick tips:

| Element | How to Modify |
|---------|---------------|
| **Colors** | Edit the `:root` variables or change values in `body` and `.card` |
| **Pricing** | Modify the `.amount` spans in each card |
| **Features** | Add/remove `<li>` elements in the `.features` list |
| **Fonts** | Change the Google Fonts link in `<head>` |
